# @caiohperlin/calc-rn

Calculadora simples em React Native (Expo + TypeScript) com as operações:

- Soma (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)

## Requisitos

- Node.js instalado
- pnpm instalado

## Como executar

```bash
cd /home/runner/work/ext/ext/calc-rn
pnpm install
pnpm start
```

Depois, escolha uma plataforma:

- `a` para Android
- `i` para iOS (macOS)
- `w` para Web
